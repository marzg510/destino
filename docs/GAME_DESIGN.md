## コンセプト

### タイトル
**Destino（デスティノ）**
- 意味: スペイン語で「運命」
- 運命は与えられるものではなく、自分の手で紡ぐもの
- 小さな行動の積み重ねが、大きな運命を創る

### ジャンル
癒しの放置ゲーム / マインドフルネス体験

### ターゲット
- 疲れている人
- 不安を抱えている人
- モヤモヤした気持ちの人
- 心の休息を求めている人

### プレイ時間
1セッション: 5〜10分（推奨）
- 短い時間で癒しを得られる
- いつでも開いて、いつでも閉じられる
- 続けることで世界が成長する

## 核心のメッセージ

### このゲームが伝えたいこと

**「幸せは、今この瞬間の中にある」**

- 幸せは未来にあるものではない
- 過去にあったものでもない
- 今、ここにある

**「生きる意味は、幸せになること」**

- 特別なことをしなくていい
- 遠くへ行く必要もない
- 今、ここにある美しさに気づくこと

**「小さな行動が、大きな幸せの循環を生む」**

- 一人では小さくても
- つながれば大きな力になる
- 優しさは循環する

**「不安を消さなくていい。ただ、今ここに在る」**

- 不安は消さなくていい
- 戦わなくていい
- ただ、今この瞬間に戻ってくる
- それだけでいい

## テーマ

### 1. 引き算の美しさ

世界は既に美しい。ただ、それが見えなくなっているだけ。

- 余計なものを取り除くと、本来の輝きが現れる
- 足すことではなく、引くことで気づく幸せ
- シンプルであることの価値

### 2. つながりの実感

あなたの小さな行動が、誰かの笑顔につながる。

- 一人では小さくても、つながれば大きな力
- 孤独ではなく、みんなでつくる世界
- 役割分担と協力

### 3. 循環する優しさ

ゴミを拾う → 世界がきれいになる → 人々が喜ぶ → 笑顔が生まれる → その想いが精霊を強くする → もっときれいにできる

- 奪うのではなく、与え合う循環
- 持続可能な幸せ
- エコロジーとの調和

### 4. 誰でもできる、でも尊い

- 特別な力はいらない
- 特別な才能もいらない
- ただ、ゴミを拾う
- それだけで世界を変えられる
- 日常の中にある英雄性

### 5. 考えずに、感じる

- 戦略を立てなくていい
- 効率を求めなくていい
- 競争しなくていい
- ただ、眺める、ただ、触れる、ただ、感じる
- 忙しい日常から離れて、ただ「在る」ことの心地よさ

### 6. 生きることの意味

精霊は「想い」から生まれた。

- 嬉しい気持ち、感謝の気持ち、幸せな気持ち
- それが消えそうになっている
- でも、行動することで想いは守られる
- 生きる理由は、誰かの幸せの中にある

## ゲームシステム

### 基本的な流れ

#### 序盤（手動＋自動）

プレイヤー（精霊）がゴミを拾う
- タップしてもいい → 気持ちいい
- 放置してもいい → 癒される
- どちらでもOK

#### 中盤（仲間が増える）

自動化の理由：共感の連鎖

- ゴミを拾う姿を見て、他の生き物や人々が集まってくる
- 仲間が自然に現れる（配置は考えなくていい）
- それぞれが自動で働く

仲間の例：
- 🐦 小鳥：木の上のゴミを拾う
- 🐿️ リス：木の周りのゴミを拾う
- 🐕 犬：広範囲を走り回る
- 🧒 子供：人間のゴミを拾う
- 👨👩 大人：組織的に活動
- 🌬️ 風：自然の力（エコテクノロジー）

#### 後半（世界が広がる）

フェーズの遷移：
- Phase 1: 公園（地上）
- Phase 2: 海
- Phase 3: 地球（宇宙から）
- Phase 4: 太陽系
- Phase 5: 銀河
- Phase 6: 宇宙

### 成長システム

#### 精霊の成長

レベルアップで：
- 見た目が変わる（光が強くなる）
- 能力が上がる（速度、範囲）
- 仲間の反応がある

#### 仲間との出会い

自動的ではなく、物語がある：

例：小鳥との出会い
```
木の下のゴミを10個拾う
↓
木がざわざわ...
↓
🐦 小鳥が降りてくる
↓
「ありがとう、ここが住みやすくなったよ」
「僕も手伝うね」
↓
[一緒に拾う] ボタン
↓
仲間に加わる
```

納得感と実感：
- なぜ現れたかが分かる
- 自然な流れ
- 仲間リストで確認できる
- それぞれの貢献が見える

### ゴミの増減システム

#### 生活は続く

- ゴミを拾っても、新しいゴミが出る
- 人々の生活が続いているから
- でも、「増える速度」より「減る速度」が速い
- 確実に減っている実感

#### ゴミメーター

```
現在のゴミ: 87個
増加: +2/分（人々の生活）
減少: -5/分（あなたと仲間）
結果: -3/分 「減ってきてる！」
```

#### 浄化率

```
公園: 78%（きれい✨）
海岸: 45%（まあまあ）
街: 12%（まだまだ...）
```

- 完全にゼロにはならない
- でも「きれいな状態」を保てる
- 維持することの大切さ

**Phase 3: オーラが広がる**
- 翼のようなものが見える
- 周りに花びらや光が舞う

**Phase 4: 地球規模**

- 地球を包む存在

- 金色～白金色

**Phase 6: 宇宙と同化**
- 透明な光
- すべてを包む

- ふわふわと浮遊
- ゆっくり移動
- ゴミを見つけると近づく
- 優しく触れる
- 嬉しそうに揺れる

### 性格

- 優しい
- 寄り添う
- 否定しない
- 解決しようとしない
- ただ、一緒にいる

### セリフの例

```

「......」

「なんだか懐かしい」

これらの嬉しい気持ちが集まって生まれた。

ゴミが増えると：
- 嬉しい想いが消えていく
- 精霊が弱くなる
- 消えてしまうかも...

だから、ゴミを拾う：
- 世界がきれいになる
- 人々の笑顔が戻る
- 嬉しい想いが増える
- 精霊が強くなる
- 生きていける

「自分のため」でもあり「みんなのため」でもある。

## ゴミ拾いの意味

### なぜゴミ拾いなのか

#### 1. 引き算の美しさ

多くのゲームは「足す」楽しさ。
- 建物を建てる
- 街を発展させる
- 装飾する

このゲームは「引く」美しさ。
- ゴミを取る
- 本来の姿が現れる
- シンプルになる

哲学：
「幸せは足すものじゃない。余計なものを引くことで見える」

#### 2. 既にある美しさに気づく

```
before: ゴミだらけの公園 → 「汚いな...」
after: ゴミを取り除く → 「あ、こんなに美しかったんだ」
```

世界は既に美しい。ゴミが隠しているだけ。

#### 3. 誰でもできる、でも大切なこと

ゴミ拾いの特徴：
- 特別な才能不要
- 道具も不要
- お金もかからない
- 今すぐできる

でも：
- 勇気がいる
- 継続が必要
- 目立たない
- すぐには評価されない

メッセージ：
「小さな行動が世界を変える」
「誰でもヒーローになれる」

### ゴミ拾いが役に立つ理由

#### 連鎖する効果

```
ゴミを拾う
↓
地面・水・空気がきれいになる
↓
植物が育つ、動物が戻る、人が集まる
↓
子供が遊べる、老人が散歩できる、笑顔が増える
↓
幸せが循環する
```

#### 具体例

**川のゴミを拾う：**
- 水が透明になる
- 魚が泳ぐ
- 子供が遊べる
- 散歩する老人
- みんな笑顔

**公園のゴミを拾う：**
- ベンチがきれいに
- 木に花が咲く
- 子供が遊ぶ
- 家族でピクニック
- 「気持ちいいね」

**海岸のゴミを拾う：**
- 砂浜が見える
- カニが這う
- 貝殻が見つかる
- 「貝殻拾い！」
- 「泳げる！」

### ゴミと循環

集めたゴミの使い道（エコロジー）：

```
プラスチック → バイオ燃料 → ドローンの動力
金属 → 新しい道具 → より効率的な収集
紙 → 種子ペーパー → 緑化
```

「ゴミは資源」の実感。

## マインドフルネス設計

### 根底に流れる思想

#### 表に出すもの

プレイヤーが見るもの：
- 精霊がゴミを拾う
- 世界がきれいになる
- 仲間が増える
- 花が咲く
- 音楽が流れる

シンプルで、優しくて、美しいゲーム。

#### 根底に流れるもの

設計者が込めた想い：

このゲームを開いている、その時間が自然と：
- 呼吸が深くなる
- 思考が静まる
- 今に意識が向く

マインドフルネスな時間。

でも、それを言葉にはしない。
ただ、そうなるようにデザインする。

### 実装方針

#### やること

✅ ゆっくりしたテンポ
- すべてがゆっくり動く
- 急かさない
- 結果として呼吸が整う

✅ 優しい色彩
- 水彩画のような柔らかい色
- 光と影の調和
- 見ているだけで心が落ち着く

✅ 心地いい音
- 自然音（風、波、鳥）
- 静かな音楽
- 結果として思考が静まる

✅ 美しい変化
- ゴミが消える
- 花が咲く
- 今に意識が向く

✅ シンプルな操作
- タップするだけ
- または何もしない
- 複雑さがない

✅ 押し付けないメッセージ
- 説教しない
- 指示しない
- ただ、寄り添う

#### やらないこと

❌ 「マインドフルネス」と言わない
❌ 「瞑想アプリ」と名乗らない
❌ 「呼吸法」を教えない
❌ 「今に集中しましょう」と指示しない
❌ 効果を強調しない

### 目指す体験

プレイヤーが自然と：
- 呼吸が深くなる
- 思考が静まる
- 今を感じる

でも、それを意識させない。

「気づいたら落ち着いてた」

その体験。

### 理想的なプレイ体験

```
疲れた日の夜

何となくアプリを開く
（瞑想しようとは思ってない）

優しい画面、ゆっくりした音楽

精霊がふわふわ動いてる

何となく見てる

タップする、ゴミが消える、花が咲く
「きれいだな」

気づいたら10分経ってた

「あれ？」
呼吸が楽になってる
肩の力が抜けてる
頭がすっきりしてる

「このゲーム、いいな」

理由は分からない
でも開くと落ち着く

━━━━━━━━━━━━━━━━

それが、このゲームの成功

マインドフルネスという言葉は知らなくてもいい
体験さえしてくれれば
```

「変わらない」でも：
「それでいいよ。ここにいてくれてありがとう」

結果を求めない。変化を強制しない。

```
表向きは: 「癒しのゴミ拾いゲーム」

でも、それを言葉にはしない
ただ、そう設計する

プレイヤーは「いいゲームだな」
それだけ思ってくれれば十分

ゲームという形で
優しさを届けたい

心の休息の場所を作りたい
